TripPlan

GET Get suitable trip plan
개요 요청 응답 예시

GET Get suitable trip plan

개요

여행 계획 API는 여행동안 머물 기간, 여행 시작 시각, 여행 종료 시각을 입력 받으면 입력한 기간 내에 여행 일정표를 제공해줍니다.

요청

curl --request GET \
--url "http://ip_name:port_num/tripplan/days/{days}/between/startTime/{startTime}/and/endTime/{endTime}"

요청 파라미터

여행 계획 API에 필요한 파라미터에 대한 내용입니다.

파라미터 타입 필수여부 설명
days string Y 여행하면서 머물 기간
startTime string Y 여행 시작 시각
value: 0000
  • 1,2번째 인덱스 값: hour
  • 3,4번째 인덱스 값: minute
endTime string Y 여행 종료 시각
value: 0000
  • 1,2번째 인덱스 값: hour
  • 3,4번째 인덱스 값: minute

요청 바디

  • 해당 API는 요청 바디를 필요로 하지 않습니다.

응답

응답 바디

JSON 형태로 정보들을 반환합니다.

속성 타입 필수여부 설명
code result code Y 0: 실패, 1: 성공
message string Y code에 대한 설명
result JsonObject Y summary와 tripplan 정보를 가지는 JsonObject

result-summary

속성 타입 설명
days string 여행동안 소요 일수
message string code에 대한 설명
result JsonObject summary와 tripplan 정보를 가지는 JsonObject

result-tripplan

day 1일차부터 n일차까지의 방문할 장소 정보와 장소간 이동정보 제공합니다.

속성 타입 설명
day string 여행 중 n번째 날
tourList JsonObjectArray 해당 일차의 방문할 장소 정보와 장소간 이동정보

result-tripplan-tourList 중 type = tour

속성 타입 설명
num string 순서번호
type string 현재 JsonObject가 방문할 장소 정보(tour)인지 장소간 이동 정보(move)인지 표시
name JsonObject 방문 장소 이름
lat string 방문 장소의 위도
lng string 방문할 장소의 경도
visitStart string 방문 시작 시각
visitEnd string 방문 종료 시각
addr string 방문 장소 주소
description string 방문 장소에 대한 설명

result-tripplan-tourList 중 type = move

속성 타입 설명
num string 순서번호
type string 현재 JsonObject가 방문할 장소 정보(tour)인지 장소간 이동 정보(move)인지 표시
distance string 이동 거리
duration string 이동 시간
